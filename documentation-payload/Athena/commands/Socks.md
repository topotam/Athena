## Summary
Start the SOCKS5 listener

- Needs Admin: False  
- Version: 1  
- Author: @checkymander  

### Arguments

#### path

None

## Usage

```
# Defaults to port 4444
socks start 

# Set listener port
socks start 1337

# Stop SOCKS5 listener
socks stop
```
