## Summary
Load a command into the agent
 
- Needs Admin: False  
- Version: 1  
- Author: @checkymander  

### Arguments
#### command

- Description: The command to be loaded.
- Required Value: True  
- Default Value: None  

## Usage

```
load [command]
```

## Detailed Summary
Valid commands that can be loaded
* cat
* cd
* cp
* env
* get-clipboard
* hostname
* ifconfig
* ls
* mkdir
* mv
* nslookup
* pwd
* rm
* wget
* whoami
