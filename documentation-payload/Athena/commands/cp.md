## Summary
Copy a file from one location to another.
  
- Needs Admin: False  
- Version: 1  
- Author: @checkymander

### Arguments

#### source

- Description: Source file to copy.  
- Required Value: True  
- Default Value: None  

#### destination

- Description: Location to copy to
- Required Value: True  
- Default Value: None  

## Usage

```
cp [source] [destination]
```
